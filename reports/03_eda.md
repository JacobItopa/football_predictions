# 03 Exploratory Data Analysis (EDA)

## Dataset Overview
- **19 season CSV files** from football-data.co.uk covering EPL matches from 2010/11 to 2024/25.
- **6 seasons of xG data** from Understat.com (2020/21 to 2025/26).
- After merging and cleaning: **7,189 interim rows** across all seasons.

## Descriptive Statistics

### Match Outcome Distribution (FTR)
Home wins are typically the most common outcome in the Premier League (~45%), followed by Away wins (~29%) and Draws (~26%). This class imbalance is expected and consistent with long-run EPL statistics.

### Key Columns (football-data.co.uk)
| Column | Description |
|---|---|
| `FTHG` / `FTAG` | Full-time home/away goals |
| `FTR` | Full-time result (H / D / A) |
| `HS` / `AS` | Total shots home/away |
| `HST` / `AST` | Shots on target home/away |
| `B365H/D/A` | Bet365 odds for each outcome |

### Key Columns (Understat xG)
| Column | Description |
|---|---|
| `Home_xG` | Expected goals generated by the home team |
| `Away_xG` | Expected goals generated by the away team |

## Identified Issues & Resolutions

| Issue | Resolution |
|---|---|
| **Missing Values** — Early seasons lack betting columns | Filled with `B365` median values or `AvgH/D/A` equivalents |
| **Date Formats** — `dd/mm/yy` vs `dd/mm/yyyy` inconsistency | Used `pd.to_datetime(format='mixed', dayfirst=True)` |
| **xG Coverage** — Only 6 seasons have xG data | Feature fallback: for pre-xG matches, actual goals (`FTHG`/`FTAG`) are used instead |
| **Team Name Mismatches** — Understat vs football-data.co.uk naming | Resolved via `TEAM_MAPPING` dict in `fetch_xg_data.py` |
| **Rolling Window Burn-in** — First 5 matches per team have no rolling history | Rows dropped from processed dataset; 7,020 clean rows remain |
| **xg_data.csv in raw glob** — Would cause duplication if included in merge | Explicitly excluded from file glob in `make_dataset.py` |
